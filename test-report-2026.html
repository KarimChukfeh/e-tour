Multi-Level Escalation Framework

Escalation Timeline: EL1: Enrollment Force Start (After enrollment window expires) ├─ Enrolled player can force start tournament └─ Solo player wins immediately if no others enrolled EL2: Enrollment External Claim (EL1 + 5 minutes) ├─ Any external player can claim abandoned pool └─ All enrolled players forfeit their fees ML1: Match Timeout Claim (After player time bank expires) ├─ Opponent claims victory via timeout └─ Tournament continues with timeout winner advancing ML2: Advanced Player Force Eliminate (ML1 + 2 minutes) ├─ Players who won their matches can eliminate stalled matches ├─ Both players in stalled match are eliminated (double elimination) └─ Ensures tournament progress even with multiple stalls ML3: External Player Replacement (ML1 + 4 minutes) ├─ ANY external player can replace stalled match players ├─ External player advances as winner ├─ Original players forfeit to pool └─ Blocked for advanced players to prevent abuse Critical Rule: ML2 NEVER expires - can be used anytime after activation
