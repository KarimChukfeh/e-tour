The matchLevel3Delay was incorrectly set to 2 minutes (same as L2), causing L2 and L3 to become available simultaneously. Tests were also using hardcoded timeout values instead of reading from contract configuration.

Fix Applied:
• Updated matchLevel3Delay to 4 minutes (cumulative from timeout)

• Updated EscalationHelpers tests to dynamically read timeout config
